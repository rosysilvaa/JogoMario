# Super Mario Jump - JavaScript Game 🍄⭐

Um projeto interativo e divertido que recria a mecânica clássica de pulo do Super Mario, desenvolvido para consolidar conhecimentos em lógica de programação com JavaScript e manipulação de animações com CSS.

## 🚀 Sobre o Projeto

O objetivo do jogo é simples: ajudar o Mario a pular os obstáculos (canos) que surgem em sua direção. A velocidade e o timing são essenciais para sobreviver o maior tempo possível.


## 🛠️ Tecnologias Utilizadas

Para a construção deste projeto, foram utilizadas as seguintes ferramentas:

- **HTML5**: Estruturação dos elementos do jogo e cenário.
- **CSS3**: Responsável pelas animações (`keyframes`), posicionamento e estilização visual.
- **JavaScript**: Controle da lógica do jogo, detecção de colisões e eventos de teclado.
